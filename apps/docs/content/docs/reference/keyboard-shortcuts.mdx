# Keyboard Shortcuts

data-peek is designed for keyboard-first navigation. Here are all available shortcuts.

## Command Palette & AI

| Action | macOS | Windows/Linux |
|--------|-------|---------------|
| Open command palette | `Cmd+K` | `Ctrl+K` |
| Open AI assistant | `Cmd+I` | `Ctrl+I` |

## Query Execution

| Action | macOS | Windows/Linux |
|--------|-------|---------------|
| Execute query | `Cmd+Enter` | `Ctrl+Enter` |
| Format SQL | `Shift+Alt+F` | `Shift+Alt+F` |
| Open saved queries | `Cmd+Shift+S` | `Ctrl+Shift+S` |
| Clear results | `Cmd+K` | `Ctrl+K` |

## Tab Management

| Action | macOS | Windows/Linux |
|--------|-------|---------------|
| New tab | `Cmd+T` | `Ctrl+T` |
| Close tab | `Cmd+W` | `Ctrl+W` |
| Switch to tab 1-9 | `Cmd+1` to `Cmd+9` | `Ctrl+1` to `Ctrl+9` |
| Next tab | `Cmd+Option+→` | `Ctrl+Alt+→` |
| Previous tab | `Cmd+Option+←` | `Ctrl+Alt+←` |

## Window Management

| Action | macOS | Windows/Linux |
|--------|-------|---------------|
| New window | `Cmd+Shift+N` | `Ctrl+Shift+N` |
| Close window | `Cmd+W` | `Alt+F4` |

## Connection Management

| Action | macOS | Windows/Linux |
|--------|-------|---------------|
| Switch to connection 1-9 | `Cmd+Shift+1` to `Cmd+Shift+9` | `Ctrl+Shift+1` to `Ctrl+Shift+9` |

## View

| Action | macOS | Windows/Linux |
|--------|-------|---------------|
| Toggle sidebar | `Cmd+B` | `Ctrl+B` |
| Open settings | `Cmd+,` | `Ctrl+,` |
| Toggle dev tools | `F12` | `F12` |
| Zoom in | `Cmd++` | `Ctrl++` |
| Zoom out | `Cmd+-` | `Ctrl+-` |
| Reset zoom | `Cmd+0` | `Ctrl+0` |
| Full screen | `Cmd+Ctrl+F` | `F11` |

## Editor

| Action | macOS | Windows/Linux |
|--------|-------|---------------|
| Undo | `Cmd+Z` | `Ctrl+Z` |
| Redo | `Cmd+Shift+Z` | `Ctrl+Shift+Z` |
| Cut | `Cmd+X` | `Ctrl+X` |
| Copy | `Cmd+C` | `Ctrl+C` |
| Paste | `Cmd+V` | `Ctrl+V` |
| Select all | `Cmd+A` | `Ctrl+A` |
| Find | `Cmd+F` | `Ctrl+F` |
| Trigger autocomplete | `Ctrl+Space` | `Ctrl+Space` |

## Application

| Action | macOS | Windows/Linux |
|--------|-------|---------------|
| Quit | `Cmd+Q` | `Alt+F4` |
| Hide | `Cmd+H` | - |
| Minimize | `Cmd+M` | - |

## Results Navigation

| Action | macOS | Windows/Linux |
|--------|-------|---------------|
| Navigate FK | `Cmd+Click` | `Ctrl+Click` |
| Copy cell | Click cell | Click cell |

## Data Editing

| Action | macOS | Windows/Linux |
|--------|-------|---------------|
| Save pending changes | `Cmd+S` | `Ctrl+S` |
| Discard all changes | `Cmd+Shift+Z` | `Ctrl+Shift+Z` |
| Add new row | `Cmd+Shift+A` | `Ctrl+Shift+A` |
| Exit edit mode | `Escape` | `Escape` |

## Tips

- Press `Cmd+K` / `Ctrl+K` to access the command palette for quick access to all features
- Most shortcuts work when the editor is focused
- Tab and window shortcuts work globally
- Connection shortcuts work globally
